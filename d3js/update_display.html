d3js - update display

Notes:

This is an example of a simple bar chart using d3js illustrating how the data can be dynamically updated. In this case two data arrays are provided in the javascript: dataArray1, and dataArray2. These two sets of data are then used alternately to update the display at one second intervals.

d3js provides three basic methods for dealing with changes in the data; update, enter, and exit. In this case the update method is used since the number of data entries remains the same in each array. Use of enter is also illustrated in this code. The exit method allows for removal of data.

Please see accompanying code for details.
