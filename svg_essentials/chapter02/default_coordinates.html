SVG rectangle using default units





"The world of SVG is an infinite canvas.

The area of the canvas that your document intends to use is called the viewport."

The two quotes above are taken from chapter 2 of 'SVG Essentials' by J. David Eisenberg.

This example shows a rectangle drawn in a viewport where both the viewport dimensions and those of the rectangle are specifed in default units. These are units without dimensions for which the display assumes pixels. The result is what might be expected.

Please note that the grid is shown at 20px intervals. Please see the accompanying code.
