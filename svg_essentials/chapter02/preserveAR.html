SVG - using preserveAspectRatio "alignment meet"





This example shows the attribute preserveAspectRation "alignment meet" in action in a case where the aspect ratio of the viewport and the viewBox differ. In this case the viewport is 1:3 and the viewBox 1:1.

A lightgrey background has been shown on the viewport so that the action of the attribute can be more clearly seen. Please see the accompanying code.
