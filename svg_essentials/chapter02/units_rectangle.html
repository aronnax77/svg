SVG rectangle using explicit units





"The world of SVG is an infinite canvas.

The area of the canvas that your document intends to use is called the viewport."

The two quotes above are taken from chapter 2 of 'SVG Essentials' by J. David Eisenberg.

This example shows a rectangle drawn in a viewport where the viewport dimensions are specifed in default units and the rectangle is specified using explicit dimensions in mm. In effect the explicit dimensions are converted back into default units before rendering.

Please note that the grid is shown at 20px intervals. Please see the accompanying code.
