SVG - User Coordinates





This example shows how user coordinates can be specified using the viewBox attribute. The svg viewport in this example measures 6cm x 4cm. The user coordinates are set at 2 units per centimeter. A rectangle is then drawn which is 6 x 4 user units and renders at half scale. That is 1:2.

Please note that the grid is shown at 1cm intervals. Please see the accompanying code.
