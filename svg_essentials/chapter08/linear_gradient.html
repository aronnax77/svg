SVG Linear Gradient

An example of how a linear gradient can be used. Note that gradientUnits is set to objectBoundingBox by default and that percentages or fractions must therefore be used to specify the gradine positioning. This can be changed to userSpaceOnUse if desired.

This example is based on a similar example in chapter 8 of 'SVG Essentials' by J. David Eisenberg.
