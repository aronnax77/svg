SVG Patterns 01

This is an example of the behaviour of an svg pattern applied using patternUnits="objectBoundingBox". This is the default setting when using a pattern and should be accompanied by width and height attributes set in percentages or as a decimal between 0 and 1 if a meaningful pattern is to result. In this case a setting of 0.2 will give the same result. Approaching a pattern in this way ensures that the same number of pattern images are used whatever the background size.

This example is based on a similar example in chapter 8 of 'SVG Essentials' by J. David Eisenberg.
