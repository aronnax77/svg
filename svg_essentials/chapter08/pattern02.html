SVG Patterns 02

This is an example of the behaviour of an svg pattern applied using patternUnits="userSpaceOnUse". This follows on directly from the previous example 'pattern 01'. In this case width and height of the pattern are specified in user units. Approaching a pattern in this way ensures that the pattern remains a consistent size regardless of the size of the bounding box.

This example is based on a similar example in chapter 8 of 'SVG Essentials' by J. David Eisenberg.
