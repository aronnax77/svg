SVG Patterns 03

This is an example of the behaviour of an svg pattern applied using patternContentUnits= "objectBoundingBox". The default is 'userSpaceOnUse' and will have automatically have been applied to patterns 01 and 02. In this case a single user unit corresponds to the bounding box itself and all units relating to the pattern must therefore be reduced to values between 0 and 1.

This example is based on a similar example in chapter 8 of 'SVG Essentials' by J. David Eisenberg.
