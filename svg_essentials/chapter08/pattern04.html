SVG Patterns 04

This example illustrates the use of a viewBox within the specification for the pattern. It is also possible to use preseerveAspectRation. As will be seen from the accompanying code a 100 x 100 svg image is reduced to 20 x 20 and used as the pattern element.

This is based on a similar example in chapter 8 of 'SVG Essentials' by J. David Eisenberg.
