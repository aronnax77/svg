SVG spreadMethod

This example shows the effect of using the spreadMethod attribute. This can be set to pad (which simply fills the remaining space with the last stop colour), repeated (which repeates the pattern in the space available), and reflected (which reverses the gradient and then fills the remaining space).

This example is based on a similar example in chapter 8 of 'SVG Essentials' by J. David Eisenberg.
