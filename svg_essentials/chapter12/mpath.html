SVG mpath and rotate="auto"

An example of the use of mpath and rotate="auto" in a complex animation. The use of mpath allows a reference to be made to an existing path and so avoids repetition of code. rotate="auto" ensures that the moving object remains aligned with the motion path. Please contrast this example with the previous example 'SVG complex motion along a path' at https://code.sololearn.com/Wjgd5zFy4y8U/#html

This example is based on a similar example in chapter 12 of 'SVG Essentials' by J. David Eisenberg.
